## Installation

```bash
# create a new virtualenv
virtualenv --no-site-packages pyenv
source ./pyenv/bin/activate

# install dependencies
pip install -r requirements.txt
```


## Usage

Run the jupyter notebook:

```
./pyenv/bin/jupyter notebook
```
